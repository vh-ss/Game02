# 🦊 Лісові Квести (Forest Quest)

Топ-даун пригодницька гра про тварин у мультяшному стилі. Один самодостатній HTML-файл — **уся графіка та звук генеруються кодом**, тож гра працює повністю офлайн, без жодних зовнішніх ресурсів.

Ти — лисичка Руда. У великому чарівному лісі біда, а ще завелися дикі звірі. Допоможи кожному другові — і врятуй лісове свято!

## 🎮 Як грати

Просто відкрий [`index.html`](index.html) у будь-якому сучасному браузері (двічі клікни на файл).

### Керування (ПК)
| Клавіша | Дія |
|---|---|
| `W A S D` / стрілки | рух |
| `Space` | удар хвостиком |
| `E` | дія / говорити / гортати діалог |
| `Shift` | понюхати (показує напрямок до предмета) |
| `Q` | журнал квестів |

### Мобільний
На сенсорних пристроях автоматично з'являється віртуальний джойстик і кнопки (Удар, Дія, Нюх, Журнал).

## ✨ Можливості

- **Великий ліс** (72×56 тайлів) з озером, скелями, лугом, стежками
- **12 завдань** у 7 персонажів, різні типи квестів:
  - 🐿️ **Збирання** — жолуді, ягоди, квіти
  - 🐻 **Перемога** — прогнати ос-злодійок
  - 🦫 **Крафт** — принести ковалю руду й дерево, він викує ключ
  - 🐰 **Визволення** — врятувати малюка з клітки
  - 🦉 **Бій з босом** — подолати Кажанище
  - 🦌 **Комбіновані** — бій + збирання в одному квесті
  - 🦔 **Фінал** — прикрасити святкову арку
- **Бойова система**: удар хвостиком, здоров'я-сердечка, регенерація
- **Небезпечні хижаки**: вовки 🐺 (лишають ікла) та кабани 🐗 бродять лісом
- **Мультяшна графіка**, намальована на canvas, та звук через WebAudio
- **Контекстний компас** і журнал квестів

## 🛠️ Технології

Чистий HTML5 Canvas + JavaScript. Без бібліотек, без збірки, без залежностей.
