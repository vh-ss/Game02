# ForestQuest — «Лісові Квести»

Браузерна гра.

## Що це
- **Один HTML-файл**: `index.html` (гра в одному файлі, запускається у браузері).
- `manifest.json` — PWA-маніфест.

## Межі проєкту
Працювати ТІЛЬКИ в цій папці (`TEST\ForestQuest`). Інші проєкти в `TEST` не чіпати.

## Git
- Репозиторій: **vh-ss/Game02**, гілка `main`.
- Після кожної зміни — самостійно `commit` + `push`.
- Push працює лише з `http.sslBackend=schannel` (виставлено глобально).

## Робота
- Без збірки — редагуєш `index.html` напряму.
- Перевірка: відкрити `index.html` у браузері.
